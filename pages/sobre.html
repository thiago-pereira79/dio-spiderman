SOBRE O PROJETO

Este multiverso do Homem-Aranha foi adaptado por Thiago Pereira para estudar na prática alguns conceitos importantes de front-end:

  • Animações e transições em CSS;
  • Criação de um carrossel 3D usando transformações;
  • Organização de páginas internas para cada versão do herói;
  • Integração com vídeos e galerias de imagens;
  • Experiência de navegação entre diferentes filmes dentro do mesmo layout.

A ideia não é só reproduzir o layout original, mas criar uma versão personalizada, com estrutura própria de pastas, textos e melhorias de usabilidade, como botões de retorno, navegação entre universos e uma página dedicada para esta descrição.
